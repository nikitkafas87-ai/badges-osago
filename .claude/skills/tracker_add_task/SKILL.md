# Яндекс Трекер — создание задачи

Команда `/tracker_add_task` создаёт задачу в Яндекс Трекере.

## Параметры

- **summary** — заголовок задачи (обязательно)
- **assignee** — исполнитель
- **queue** — очередь (по умолчанию `INS`)
- **deadline** — дедлайн в формате ISO 8601
- **description** — описание задачи
- **priority** — приоритет от `critical` до `minor` (по умолчанию `normal`)

## Алгоритм

1. **Разбор параметров** — извлечь данные из запроса. Если заголовок не указан — спросить перед созданием.

2. **Разрешение исполнителя** — найти пользователя. Для новых аккаунтов использовать числовой ID, для старых — логин.

3. **Создание задачи** — вызвать create_issue со всеми параметрами. Дедлайн передаётся напрямую — отдельный update не нужен.

4. **Подтверждение** — вернуть ссылку на задачу (`https://tracker.yandex.ru/<KEY>`), имя исполнителя, дедлайн и очередь.

## Правила

- Очередь по умолчанию — `INS`, если не указана другая
- Текстовые даты (например, «11 марта») конвертировать в формат YYYY-MM-DD
- Никогда не создавать задачу без заголовка
